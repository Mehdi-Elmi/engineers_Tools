# قوانین تعامل محیط طراحی

این فایل قانون‌های تعامل فعلی برای محیط `Engineering Design Tools` را ثبت می‌کند.

## مسیر فعال

```text
modules/mechanics_dynamics_statics/workspace.py
modules/mechanics_dynamics_statics/interaction_fixes.py
```

`interaction_fixes.py` فعلاً اصلاحات کم‌ریسک interaction را روی محیط فعال نصب می‌کند تا فایل‌های بزرگ و پایدار پروژه بی‌دلیل بازنویسی نشوند.

## قانون Resize

- صفحه یا canvas نباید به خاطر resize آبجکت تغییر اندازه واقعی بدهد.
- resize آبجکت باید با anchor درست انجام شود؛ یعنی وقتی کاربر از لبه چپ می‌کشد، لبه راست باید مرجع بماند و برعکس.
- گوشه‌ها و بالا/پایین باید hit-test بزرگ‌تر و قابل گرفتن داشته باشند.
- حداقل اندازه آبجکت باید رعایت شود، اما clamp نباید باعث جابه‌جایی نادرست anchor شود.

## قانون Cursor و Indicator

- از cursorهای خام Qt مثل `SizeHorCursor`, `SizeVerCursor`, `SizeFDiagCursor` به عنوان ظاهر نهایی استفاده نشود.
- cursorهای move، rotate، resize و ruler guide باید از الگوی گرافیکی پروژه پیروی کنند.
- این قانون بعداً برای ابزارهای line، snap، rotate، magnet و constraint هم باید مبنا باشد.

## قانون Ruler و Zoom

- ruler باید با zoom فعلی canvas هماهنگ باشد.
- وقتی zoom تغییر می‌کند، فاصله tickهای ruler باید با همان scale بصری canvas هماهنگ شود.
- origin خطکش باید به همان مختصات بصری grid اشاره کند و از canvas جداگانه رفتار نکند.
- فعال کردن صفر-صفر خطکش از گوشه نباید باعث تغییر اندازه صفحه شود.

## قانون نمایش زاویه

- هنگام rotation، مقدار زاویه باید نزدیک handle دوران نمایش داده شود.
- زاویه نمایش داده‌شده باید از مبنای صفر محاسبه شود.
- واحد زاویه در UI فعلی به صورت `deg` نمایش داده می‌شود تا با کد ASCII و کنترل‌های فعلی سازگار بماند.

## قانون F8

`F8` میانبر پیش‌فرض Orthogonal Snap است.

رفتار فعلی:

- اگر آبجکت انتخاب‌شده rotation داشته باشد، با `F8` به نزدیک‌ترین محور ۰، ۹۰، ۱۸۰ یا ۲۷۰ درجه snap می‌شود.
- اگر آبجکت انتخاب نشده باشد، فقط StatusBar پیام راهنما نشان می‌دهد.
- این رفتار بعداً باید برای ابزارهای رسم مثل Line و Vector هم استفاده شود تا خط در نزدیکی محورهای XY افقی یا عمودی شود.

## قانون Properties

Properties باید `Shortcut Keys` و `F8: Orthogonal Snap` را به عنوان گزینه پیش‌فرض محیط طراحی نشان دهد.

## نکته معماری

این فایل جایگزین مستند اصلی `docs/window_pattern.md` نیست. اگر این قانون‌ها نهایی شدند، باید بعداً از حالت patch runtime خارج شوند و به کد اصلی `start_bar.py` و `workspace.py` منتقل شوند.
